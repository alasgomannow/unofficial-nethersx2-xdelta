# NetherSX2 XDelta Unofficial

> **Unofficial repository**. Not affiliated with the NetherSX2 developers.  
> This repository provides xdelta patches generated from publicly released NetherSX2 APKs.

---

## Mission

NetherSX2 stopped providing xdelta patches in recent releases. This repository exists to **restore what was left out**:  

- Reduce bandwidth usage (up to **77% savings**) by patching against **AetherSX2** instead of downloading full APKs.  
- Provide **xdelta patches for every new NetherSX2 release**, so users can keep up-to-date efficiently.  

---

## Usage

1. **Download AetherSX2 APK**  
   Get the base APK from [AetherSX2 releases](https://github.com/Trixarian/NetherSX2-patch/releases/download/0.0/15210-v1.5-4248.apk).  

2. **Download the xdelta patch**  
   Find the latest xdelta patch [here](https://github.com/alasgomannow/unofficial-nethersx2-xdelta/releases).  

3. **Apply the patch with UniPatcher**  
   - Open UniPatcher.  
   - Select the **xdelta patch** as the patch file.  
   - Select **AetherSX2 APK** as the base file.  
   - Choose an output folder.  
   - Click **Save**.  

4. **Verify integrity**  
   - After patching, compare the **SHA256** of your generated APK with the official NetherSX2 release.  
   - Copy the SHA256 from the [NetherSX2 release page](https://github.com/Trixarian/NetherSX2-patch/releases).  
   - Use any SHA256 verification tool or file manager to ensure your patched APK **matches exactly**.  

> Only after verification can you be certain your APK is identical to the official release. This ensures **safety and trust**, without distributing modified APKs.
